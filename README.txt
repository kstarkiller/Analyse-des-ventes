INTRODUCTION
Ce projet m'a permis de réaliser une analyse des données de ventes d'une librairie en ligne fictive.
J'ai tout d'abord nettoyé et réorganisé les données puis je les ai analysées.
Enfin, il était demandé de tester 5 corrélations entre l'âge des clients, leurs genres et d'autres éléments comme le panier moyen ou la catégorie des produits achetés.

ORGANISATION
3 dossiers :
- Sources : contient les données initiales du projet.
- Export : contient les données cleansées.
- P4_04_Graphiques : contient l'ensemble des graphiques du projet créés pour ce projet

3 Jupyter Notebooks :
- P4_01_ScriptNettoyage.ipynb : récupère les données du dossier Sources, les nettoie, et enregistre le résultat dans le dossier Export.
- P4_02_ScriptAnalyse : récupère les données du dossier Export, les analyse et produit des graphiques enregistrés dans le dossier Graphiques.
- P4_03_ScriptCorrélations : récupère les données du dossier Export, test les corrélations demandées et produit des graphiques enregistrés dans le dossier Graphiques.

PROCEDURE
Pour lancer les notebooks, 2 méthodes :
1. Lancer la commande jupyter-notebook dans une console positionné dans le dossier parent.
2. Consulter les notebooks directement sur Github, via ce lien : https://github.com/kstarkiller/Analyse_des_ventes
